if robotframework says oserror errno 8 exec format error it means you have installed the ARM or 64BIT version of the geckodriver on your 32BIT ed/x/ubuntu

```
robotframework OSError: [Errno 8] Exec format error
or
geckodriver: cannot execute binary file: Exec format error
```
