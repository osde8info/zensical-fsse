oracle linux 7 install from 460M CD instead of 4G DVD

- download oracle linux 7 boot iso image (460M network install CD) from edelivery
- boot from CD
- enable network card
- change repo to http://yum.oracle.com/repo/OracleLinux/OL7/latest/x86\_64
- choose a config ie server gui
- click install
