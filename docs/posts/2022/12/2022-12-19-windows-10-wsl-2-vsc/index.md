M$ Windows 10 WSL 2 M$ VSCode & Git

Install WSL, VSCODE & Git for Windows

then in your WSL instance

```
git config --global credential.helper \ 
"/mnt/c/Program\ Files/Git/mingw64/bin/git-credential-manager-core.exe"
```
