[Robot Framework](http://robotframework.org/) 5.0 _is a big new major release with lot of interesting new features that were prioritized based on the [community survey](https://github.com/pekkaklarck/rf5survey). Biggest enhancements are `TRY/EXCEPT`, `WHILE`, inline `IF`, `RETURN`, `BREAK` and `CONTINUE` syntax, support for custom argument conversion in libraries and various enhancements to xUnit outputs. Robot Framework 5.0 only works with Python 3.6 or newer._
