how to create a @dell #windows #recovery #usbdrive

note : you will need the dell TAG number and you may need TWO dell computers if you cant boot into the one you want to recover

- https://www.dell.com/support/kbdoc/en-uk/000123667/how-to-download-and-use-the-dell-os-recovery-image-in-microsoft-windows
- https://www.dell.com/support/kbdoc/en-uk/000123667/how-to-download-and-use-the-dell-os-recovery-image-in-microsoft-windows#bs\_FAQ-2
- https://www.dell.com/support/kbdoc/en-uk/000177401/restore-your-system-using-dell-supportassist-os-recovery

_**Note:** The Dell operating system recovery image must be used on Dell computers only. It is not designed or tested for use on non-Dell computers._

after you have created the usb recovery media

1. _Connect the USB recovery media to the Dell computer where you want to install Microsoft Windows 10 or Windows 11._
2. _Restart the computer._
3. _At the Dell logo screen, tap the F12 key until you see the **Preparing one time boot menu** in the top-right corner of the screen._
4. _On the boot menu, under UEFI boot, select the USB recovery drive and press the **Enter** key._
5. _On the **Choose an option** screen, click **Troubleshoot** and then click **Recover from a drive**._
