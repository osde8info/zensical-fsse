ms vscode workspace trust

_Visual Studio Code takes security seriously and wants to help you safely browse and edit code no matter the source or original authors. The Workspace Trust feature lets you decide whether your project folders should allow or restrict automatic code execution._

- https://code.visualstudio.com/docs/editor/workspace-trust
