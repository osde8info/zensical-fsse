MS #visualstudio online seems to have been replaced with MS @github #codespaces ! #webdev #ide #onlineide

Announced in 2019 it no longer seems to exist

- https://devblogs.microsoft.com/visualstudio/announcing-visual-studio-online-public-preview/
- NOW BROKEN - https://docs.microsoft.com/en-us/visualstudio/codespaces/overview/what-is-vsonline

it seems to have been replaced with github codespaces

- https://visualstudio.microsoft.com/services//github-codespaces/
- https://github.com/features/codespaces/signup?utm\_source=visualstudio.microsoft.com&utm\_medium=ref
- https://visualstudio.microsoft.com/dev-essentials/
- https://my.visualstudio.com/Benefits?mkt=en-us

but codespaces seems to ONLY be available for _"organizations using GitHub Team or GitHub Enterprise Cloud. For more information, see "[GitHub's products](https://docs.github.com/en/articles/github-s-products)."_

learn more at

- https://techcommunity.microsoft.com/t5/microsoft-learn/ct-p/MicrosoftLearn
