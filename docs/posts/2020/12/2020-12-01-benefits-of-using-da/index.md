benefits of using @dashnex with @cloudflare are you CAN point to a custom domain and you get to use the following features

Automatic HTTPS Rewrites: ON

Always use HTTPS: ON

Auto Minify: JS, CSS, HTML

Brotli: ON

SSL/TLS Recommender: ON
